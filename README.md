# graphql-code-generator usage

## How to use

* git clone graphql server api

```code
git clone https://github.com/rongfengliang/harborapi2graphql.git
```

* start graphql server

```code

cd harborapi2graphql
yarn && yarn start
```

* generate code

```code
yarn && yarn gen
```

* typings

```code

typings/types.ts
```
